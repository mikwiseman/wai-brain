# Infra Templates

This directory contains public-safe templates for wiring a private deployment.

Files here must use placeholders only. Do not commit real server addresses, chat IDs, deploy-key paths, OAuth usernames, token locations, or private workspace names.

## Files

- `openclaw/wai-brain-capture.md` — generic silent-capture skill for OpenClaw-style Telegram ingestion.

## Deployment Rule

Copy templates from this public repo into a private vault or private deployment repo, then fill in real values there. The public repo remains generic.
